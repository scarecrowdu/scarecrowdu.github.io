插件安装

  1. 在Sublime Text 3中按下快捷键Ctrl+Shift+P

  2. 在出现的文本框中输入Install Package(或直接输入“ip”)选中Install Package并回车
    （可能需要等待几秒钟，在此过程中不能进行任何鼠标/键盘操作）

  3. 输入或选择你需要的插件再按Enter（回车）就可以安装插件了（下图以安装Emmet插件为例）

  4. 在安装过程中，左下角会显示正在安装的提示，静候片刻

  5. 若安装成功，相应的，左下角会出现安装成功的提示

常用插件

  1. Emmet插件： 让编写HTML代码变得极其简单高效（基本用法：输入标签简写形式，然后按Tab键关于Emmet的更多介绍，请查看官方文档这份速查表，可以帮你快速记忆简写形式）

  2. JsFormat插件：JS格式化的插件（即可在JS文件中通过鼠标右键->JsFormat或键盘快捷键Ctrl+Alt+F对JS进行格式化）

  3. HTML-CSS-JS Prettify插件：这是一款集成了格式化（美化）html、css、js三种文件类型的插件

  4. AutoFileName：自动完成文件名的输入，如图片选取（输入”/”即可看到相对于本项目文件夹的其他文件）

  5. jQuery： jQ函数提示 （快捷输入jQ函数，是偷懒的好方法）

  6. DocBlock：生成优美注释（使用：输入/、/*然后回车）

  7. MaterialTheme：推荐主题
